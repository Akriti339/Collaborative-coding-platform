# CollabCode

A collaborative online judge built using the MERN stack, Socket.IO, Docker, and MongoDB.

## Features

- Real-time collaborative coding
- Room-based code sharing
- C++ and Python execution
- Docker sandbox execution
- Time Limit Exceeded (TLE)
- Memory Limit Exceeded (MLE)
- Compile Error (CE)
- Runtime Error (RE)
- Problem Set
- Monaco Code Editor
- MongoDB persistence
- Automatic room saving

## Tech Stack

Frontend:
- React
- Monaco Editor
- Socket.IO Client

Backend:
- Node.js
- Express
- Socket.IO
- MongoDB Atlas
- Docker
